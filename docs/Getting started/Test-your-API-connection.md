# Test your connection to the API

- The URL that you would like to request
- Your API key
- Your Authorization header

    "headers": {

    "Accept": "application/vnd.whispir.workspace-v1+json"

    "x-api-key": "OmfEAq1Lak5tjVEi9m0dC9M4iLutkHpG8OSDEYnB"
